# Delivery Handoff

1. Confirm the outgoing owner's acceptance criteria are evidenced or explicitly incomplete.
2. Produce `company/templates/agent-report.md` and link the governing Work Order and decisions.
3. Summarize changed artifacts, verified behavior, assumptions, failures, unresolved questions, and approval status.
4. Identify the next owner and transfer write ownership explicitly; never leave two active writers on the same paths.
5. Give the next owner only the context needed for its bounded task and define new acceptance criteria when scope changes.
6. Escalate scope expansion or a new approval gate to `chief_of_staff` instead of silently continuing.
7. Update operational knowledge only after the receiving owner accepts the handoff.
